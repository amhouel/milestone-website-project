my portfolio

Genius Music React Application

This project is one page web application built with React.js, styled with CSS and using data from the Genius API. The Genius Music web application allows users to search songs by Artist names to learn more about the song and listen to it via through providers like Spotify, Youtube and SoundCloud.

My Vinyl Collection Full Stack React Application

My Vinyl Collection is a full-stack web application built with React.js - styled with CSS - Node.js & Express and PostgreSQL database & Sequelize. This web application is for vinyl collectors, who can create a profile and share their vinyl collection by adding albums to their profile using our database. By clicking on an album, each collector learn more about the album and can also see who owns the album and browse other collectors' collections. For this project the database was hardcoded with a few albums.

Give Back NYC Full Stack React Application

Give Back NYC is an application for volunteers in NYC to find volunteering opportunities in their community and to log in their experiences and rate them. Volunteers get points for each rated experience. This is a full-stack, responsive, web application built with ReactJS, CSS flexbox and grid on the front-end, and NodeJS, Express and PostgreSQL paired with the Sequelize ORM on the back-end. I also used packages like bcrypt and json web token for registration and login authentication, as well as cloudinary react for the user's profile picture upload. This application also uses the Google Geolocation API for the "Near You" search feature.
