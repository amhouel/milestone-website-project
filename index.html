my portfolio

Genius Music

A React Application with the Genius API

The Genius Music web application allows users to search songs by Artist names to learn more about the song and listen to it via through providers like Spotify, Youtube and SoundCloud.

My Vinyl Collection

A PostgreSQL, Express, React, Node Application

This web application is for vinyl collectors, who can create a profile and share their vinyl collection by adding albums to their profile using our database. By clicking on an album, each collector learn more about the album and can also see who owns the album and browse other collectors' collections. For this project the database was hardcoded with a few albums.

Give Back NYC

A PostgreSQL, Express, React, Node Application

Give Back NYC is an application for volunteers in NYC to find volunteering opportunities in their community and to log in their experiences and rate them. Volunteers get points for each rated experience.
